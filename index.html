DHS — 동·호수·매물정보 정리

네이버 부동산에서 동·호수와 매물 특징 정보를 정리해 주는 Chrome 확장

최신 버전: 확인 중…

확장 다운로드 (.zip)

한 번 설치하면, 이후 새 버전은 자동 업데이트됩니다 (아래 자동 업데이트 설치 참고).

설치 방법 (2분)

  1. 받은 DHS-Extension.zip의 압축을 풉니다.
  2. Chrome 주소창에 chrome://extensions 입력 → 이동
  3. 오른쪽 위 개발자 모드 켜기
  4. 압축해제된 확장 프로그램을 로드 클릭 → 압축 푼 DHS-Extension 폴더 선택
  5. 네이버 부동산 단지 페이지를 열고 매물을 클릭하면 오른쪽 아래에 진단 오버레이가 나타납니다.

Node.js/npm 없이 브라우저만으로 동작합니다.

자동 업데이트

새 버전이 나오면 확장 아이콘에 빨간 NEW 배지가 뜹니다. 자동 업데이터를 설치하면 백그라운드에서 최신 버전을 받아 다음 Chrome 재시작 시 자동 반영됩니다.

자동 업데이터 설치 방법은 곧 여기에 추가됩니다.

문제가 있으면 배포자에게 문의하세요.
