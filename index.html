최신 버전: 확인 중…

설치 (권장) 자동 업데이트 포함

설치 스크립트를 받아 한 번 실행하면 확장이 설치되고, 이후 새 버전은 백그라운드에서 자동으로 받아져 Chrome 재시작 시 반영됩니다.

Windows 설치 (.exe)   macOS 설치

  • Windows: 받은 DHS-Setup.exe 실행 (서명 없어 SmartScreen 경고 시 '추가 정보 → 실행'). 설치가 확장 + 자동 업데이터를 설정하고, 끝나면 안내 폴더 %USERPROFILE%\DHS-Anything\DHS-Extension를 Chrome에 로드.
  • macOS: 받은 install-dhs-mac.command를 우클릭 → 열기 (보안 경고에서 '열기'). 설치 후 안내 폴더 ~/DHS-Anything/DHS-Extension를 Chrome에 로드.
  • Linux: 터미널에서 curl -fsSL https://raw.githubusercontent.com/minoTrey/DHS_Anything-dist/main/install-dhs-linux.sh | bash

Chrome에 로드 (최초 1회)

  1. Chrome 주소창에 chrome://extensions 입력 → 이동
  2. 오른쪽 위 개발자 모드 켜기
  3. 압축해제된 확장 프로그램을 로드 클릭
  4. 폴더 선택: ~/DHS-Anything/DHS-Extension (설치 스크립트가 안내) 또는 수동으로 압축 푼 DHS-Extension 폴더
  5. 네이버 부동산 단지 페이지에서 매물을 클릭하면 오른쪽 아래 진단 오버레이가 나타납니다.

Node.js/npm 없이 브라우저만으로 동작합니다.

수동 설치 (자동 업데이트 없음)

스크립트 없이 확장만 쓰려면 zip을 받아 압축을 풀고 위 방법으로 로드하세요. 이 경우 새 버전이 나오면 아이콘에 NEW 배지가 뜨며, 갱신은 zip을 다시 받아 교체해야 합니다.
